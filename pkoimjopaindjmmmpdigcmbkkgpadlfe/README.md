# Broom Cursor Chrome Extension

This Chrome extension replaces your mouse cursor with a magical broom.

## 🧪 How to Test

1. Go to `chrome://extensions`
2. Enable Developer Mode
3. Click "Load unpacked" and select this folder
4. Visit any webpage or open `test.html` in the extension

## 🔍 Troubleshooting

- Ensure your macOS Pointer Size is set to default (`System Settings > Accessibility > Display > Pointer`)
- Try disabling Chrome hardware acceleration (`chrome://settings/system`)
- Use DevTools (Cmd+Opt+I) to confirm the `cursor` style is applied
- The fallback cursor is set to `crosshair` so you know when the broom image fails to load

Enjoy the magic 🧹
